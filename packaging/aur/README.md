# AUR Packaging

This directory contains AUR repository content prepared from the packaging work
in this repository.

Current status:

- `obs-snekstudio-source-git/` is ready to use as an AUR repository payload.
- The stable `obs-snekstudio-source` AUR package is not generated here yet,
  because there is no public `v0.1.0` tag to build from.

To publish the git package, initialize or clone the AUR repository and copy the
contents of `obs-snekstudio-source-git/` into that repo root, then push it.
